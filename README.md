# 📦 Warehouse Inventory Turnover & Demand Forecast Analytics System

An AI-powered warehouse analytics dashboard that helps businesses monitor sales, predict product demand, and optimize inventory decisions using machine learning.

Built using **Python, Streamlit, Scikit-learn, and Plotly**.

---

# 🚀 Features

### 📊 Dashboard Analytics
- Sales overview
- Inventory turnover ratio
- Sales by region and category
- Fast-moving and slow-moving products

### 📈 Demand Forecasting
- Demand prediction using **Random Forest Regressor**
- Historical vs forecast visualization
- Expected vs actual performance comparison
- Forecast accuracy metrics (MAE, RMSE, MAPE)

### 🤖 AI Inventory Insights
- ABC inventory classification
- EOQ (Economic Order Quantity)
- Safety stock calculation
- Reorder point prediction
- Sales anomaly detection

### ⚠ Inventory Risk Detection
- Stock risk prediction
- Demand risk level classification
- Product restocking recommendations

### 📥 Reporting
- Download forecast report as CSV
